# Projects

This guide will walk you through the steps of managing projects.

## View projects{#view-projects}

Once you join an organization, you can view the list of all projects in the organization.

![view-projects-byoc](/byoc/view-projects-byoc.png)

## Create a project{#create-a-project}

To create a project, you must be an [Organization Owner](./resource-hierarchy).

<Admonition type="info" icon="ðŸ“˜" title="Notes">

When you create a project, you are also a [Project Owner](./resource-hierarchy) for the project.

</Admonition>

![create-project-byoc](/byoc/create-project-byoc.png)

## Rename a project{#rename-a-project}

To rename a project, you must be an [Organization Owner](./resource-hierarchy).

<Admonition type="info" icon="ðŸ“˜" title="Notes">

Each organization contains a default project. The name of the default project cannot be modified.

</Admonition>

![rename-project-byoc](/byoc/rename-project-byoc.png)

## Delete a project{#delete-a-project}

To delete a project, you must be an [Organization Owner](./resource-hierarchy).

<Admonition type="info" icon="ðŸ“˜" title="Notes">

Each organization contains a default project. The default project cannot be deleted.

</Admonition>

![delete-project-byoc](/byoc/delete-project-byoc.png)
