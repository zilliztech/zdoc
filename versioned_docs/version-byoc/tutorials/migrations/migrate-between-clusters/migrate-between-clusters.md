# Cross-Cluster Migrations

Cross-cluster migration allows you to transfer data between Zilliz Cloud clusters, whether they reside within the same organization or across different organizations. Key use cases include:

- Migrating from a free-tier cluster to a paid plan.

- Sharing data across organizations securely.

- Reorganizing infrastructure for performance optimization.
