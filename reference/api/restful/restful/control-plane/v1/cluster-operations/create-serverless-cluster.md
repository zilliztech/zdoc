## Request

### Parameters

- No query parameters required

- No path parameters required

- Header parameters

    | Parameter        | Description                                                                               |
    |------------------|-------------------------------------------------------------------------------------------|
    | __Authorization__  | **string**<br/>|

### Request Body

```json
{
    "plan": "string",
    "clusterName": "string",
    "projectId": "string"
}
```

| Parameter        | Description                                                                               |
|------------------|-------------------------------------------------------------------------------------------|
| __plan__ | __string__  <br/>The subscription plan of the cluster to be created. Possible values are `free` and `serverless`.<br/>Possible values: "**Free**", "**Serverless**"  |
| __clusterName__ | __string__  <br/>The name of the cluster to be created. It is a string of no more than 32 characters.  |
| __projectId__ | __string__  <br/>The ID of the project where the cluster is to be created.  |

## Response

Returns an object that contains the details of the created cluster.

### Response Body

```json
{
    "code": "integer",
    "data": {
        "clusterId": "string",
        "username": "string",
        "password": "string",
        "prompt": "string"
    }
}
```

| Property | Description                                                                                                                                 |
|----------|---------------------------------------------------------------------------------------------------------------------------------------------|
| __code__   | **integer**<br/>Indicates whether the request succeeds.<br/><ul><li>`0`: The request succeeds.</li><li>Others: Some error occurs.</li></ul> |
| __data__ | __object__<br/> |
| __data.clusterId__ | __string__  <br/>The ID of the cluster affected by this operation.  |
| __data.username__ | __string__  <br/>The name of the cluster user generated by default.  |
| __data.password__ | __string__  <br/>The password of the cluster user generated by default. It will not be displayed again, so note it down and securely store it.  |
| __data.prompt__ | __string__  <br/>The statement indicating that this operation succeeds.  |

### Error Response

```json
{
    "code": integer,
    "message": string
}
```

| Property | Description                                                                                                                                 |
|----------|---------------------------------------------------------------------------------------------------------------------------------------------|
| __code__   | **integer**<br/>Indicates whether the request succeeds.<br/><ul><li>`0`: The request succeeds.</li><li>Others: Some error occurs.</li></ul> |
| __message__  | **string**<br/>Indicates the possible reason for the reported error. |
