# MetricType

This is an enumeration that provides the following constants.

## Constants{#constants}

- INVALID

    Sets the metric type to __INVALID__.

- L2

    Sets the metric type to __L2__. This applies to float vectors only.

- IP

    Sets the metric type to __IP__. This applies to float vectors only.

- COSINE

    Sets the metric type to __COSINE__. This applies to float vectors only.

- HAMMING

    Sets the metric type to __HAMMING__. This applies to binary vectors only.

- JACCARD

    Sets the metric type to __JACCARD__. This applies to binary vectors only.
