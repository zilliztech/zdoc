# ConsistencyLevel

This is an enumeration that provides the following constants.

## Constants{#constants}

- __STRONG__

    Sets the consistency level to __Strong__, with a code value of 0.

- __BOUNDED__

    Sets the consistency level to __Bounded__, with a code value of 2.

- __EVENTUALLY__

    Sets the consistency level to __Eventually__, with a code value of 3.
