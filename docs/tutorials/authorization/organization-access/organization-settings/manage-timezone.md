# Manage Timezone

The system time zone is set to where your first login occurs and applies to all time strings displayed on Zilliz Cloud.

## View the current time zone{#view-the-current-time-zone}

To view the current time zone, you can either be an [Organization Owner](./resource-hierarchy) or an [Organization Member](./resource-hierarchy).

![timezone-settings](/img/timezone-settings.png)

## Edit the time zone{#edit-the-time-zone}

To change the system time zone, click **Edit** to open the **Time Zone Settings** dialog box, and select a time zone from the drop-down list. You can also enter the name of a time zone to quickly filter the desired time zone.
