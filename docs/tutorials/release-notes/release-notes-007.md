# Release Notes (Sept 15, 2022)

We're delighted to announce that Zilliz Cloud's latest significant update is now generally available. This release includes enhancements to both the collection view and the vector search interface. Additionally, it introduces the convenience of signing up via Google and offers new system maintenance settings for improved administrative control.

- Upgrades the collection view UI.

- Upgrades the vector search UI.

- Supports signup with Google. For details, refer to [Register with Zilliz Cloud](./register-with-zilliz-cloud).

- Supports system maintenance settings. For details, refer to [Set up Maintenance Window](./setup-maintenance-window).
