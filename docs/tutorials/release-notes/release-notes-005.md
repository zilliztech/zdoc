# Release Notes (Aug 1, 2022)

The Zilliz Cloud update enhances the user experience by enabling comprehensive collection management directly through the Cloud UI. Users can now view, create, and delete collections with ease, along with viewing their schemas. Additionally, the ability to view indexes within the UI has been added. These features, coupled with improvements to the Cloud UI for smoother interaction, streamline database management and ensure a more intuitive user experience.

- Supports viewing collections on Cloud UI.

- Supports viewing collections schema on Cloud UI.

- Supports creating collections on Cloud UI.

- Supports deleting collections on Cloud UI.

- Supports viewing index on Cloud UI.

- Improves Cloud UI for better interaction.
