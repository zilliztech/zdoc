# Release Notes (July 22, 2022)

Zilliz Cloud is a cloud-native service for [Milvus](https://milvus.io/), you can set up your Milvus cluster to unlock high-performance similarity search with no extra effort needed for infrastructure management. We are currently in preview and if you have any questions, please contact us at [support@zilliz.com](mailto:support@zilliz.com).

Release date: 2022-07-22

### **New features**{#new-features}

- Supports creating a vector database on AWS region: `us-west-2 Oregon`.

- Supports all Core Milvus features.

- Supports suspending and resuming vector databases.

- Supports viewing basic vector database metrics.

- Supports database user management.

- Supports creating multiple projects.

- Supports setting IP Whitelist at the project level.

- Supports viewing user operational events.

- Supports enabling MFA with email.
