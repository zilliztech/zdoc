# On Zilliz Cloud Console

The Zilliz Cloud console is an intuitive command center for you to manipulate almost everything on Zilliz Cloud. This page aims to provide tips on finding the information you need on the Zilliz Cloud console. 

## Overview{#overview}

Once you have set up a cluster on Zilliz Cloud, this is the Zilliz Cloud console that always greets you upon your login.
