### What can I do if I forget the password used to connect to my Zilliz Cloud cluster?{#what-can-i-do-if-i-forget-the-password-used-to-connect-to-my-zilliz-cloud-cluster}

If you forget your password, you can create another user with a new password. To create a new user, go to the Zilliz Cloud console. Click on the database you want to connect to. Navigate to the **Users** tab. Click on the **+ User** button as shown in the screenshot below.
