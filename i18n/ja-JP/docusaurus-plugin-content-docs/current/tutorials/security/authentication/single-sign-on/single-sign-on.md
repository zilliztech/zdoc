# シングルサインオン（SSO）

シングルサインオン（SSO）は、ユーザーが複数のアプリケーションまたはサービスに1つの資格情報セットでログインできるようにする機能で、各サービスごとに別々のログインを要求する代わりになります。

Zilliz Cloudでは、OpenID Connect（[OIDC](https://help.okta.com/en-us/content/topics/apps/apps_app_integration_wizard_oidc.htm)）または[SAML 2.0](https://help.okta.com/en-us/content/topics/apps/apps_app_integration_wizard_saml.htm)プロトコルを介してSSOを有効にすることができます。この機能は組織レベルで動作します。
