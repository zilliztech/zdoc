# 前払いを使用

Zilliz Cloudでは、クレジットカードを追加するかAWSマーケットプレイスで登録する代わりに、前払い（銀行振込）を支払い方法として受け付けています。

## 前払い資金を追加\{#add-funds-to-advance-pay}

現在、前払い残高に資金を追加するには[お問い合わせ](https://zilliz.com/contact-sales)する必要があります。

<Admonition type="info" icon="📘" title="注意">

<p>支払い方法の優先順位は：クレジット &gt; 前払い &gt; クレジットカード/AWSマーケットプレイス登録です。</p>
<p>これは、残りのクレジットがある場合、資金を追加してクレジットカードを追加した場合やAWSマーケットプレイスで登録した場合、使用料金をカバーするためにまずクレジットが差し引かれることを意味します。クレジットが十分でない場合、前払い資金が差し引かれます。資金とクレジットがまだ料金をカバーするのに十分でない場合、クレジットカードまたはAWSマーケットプレイスアカウントに請求します。</p>

</Admonition>

## 前払い履歴を表示\{#view-advance-pay-history}

銀行振込履歴を表示するには、上部ナビゲーションバーまたは左側のナビゲーションペインの請求をクリックしてください。次に、前払いセクションの履歴をクリックします。**銀行振込履歴**ページで、振込を行った時間、追加した資金額など、過去のすべての振込の詳細を表示できます。

![add-fund-en](/img/add-fund-en.png)

## 前払い残高のモニタリングを設定\{#set-monitor-for-advance-pay-balance}

デフォルトでは、前払い残高のモニタリングは無効になっています。ただし、モニタリングを有効にすると、前払い残高がモニタリング条件で指定した金額を下回ったときに通知を受け取ることができます。詳細については、[組織アラートを管理](./manage-organization-alerts)を参照してください。

## 前払い資金の返金\{#refund-advance-pay-funds}

現在、Zilliz Cloudではウェブコンソールでの返金はサポートされていません。返金するには、お問い合わせいただき、Zilliz Cloud[サポートポータル](https://support.zilliz.com/hc/en-us)でリクエストを送信してください。
