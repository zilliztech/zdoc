# コストの理解

このページでは、Zilliz Cloudの利用コストを見積もるためのガイドラインを提供しています。正確な見積もりを得るためには、[無料トライアル](./free-trials)を利用することをお勧めします。

請求書の詳細な内訳と解釈については、「[インボイス](./view-invoice)」を参照してください。

Zilliz Cloudは組織レベルで料金を請求します。組織内に複数のクラスターがある場合、使用状況に基づいてDedicatedクラスターとServerlessクラスターごとに請求され、バックアップやパイプラインのコストなどの追加費用も請求されます。

## 専用クラスター{#dedicated-clusters}

専用クラスターは、専用のリソースと環境を提供し、カスタマイズ可能なワークロード管理による強化されたパフォーマンスを提供します。

専用クラスタはCUの使用量に基づいて請求されます。その総コストは**CUコストとストレージコストの合計**です。ストレージコストは通常、総コストの約2%しか占めません。正確な見積もりについては、[計算機](https://zilliz.com/jp/pricing#calculator)を使用してください。このセクションでは、主に専用クラスタのCUコストに焦点を当てています。

#### コスト計算{#cost-calculation}

```plaintext
CU Cost = CU Unit Price x CU Number x Cluster Runtime
```

注:クラスターランタイムとは、クラスターの持続時間を指します。クラスターが「作成中」、「一時停止中」、「再開中」、「一時停止中」などの特定の状態にある場合、これらの状態で費やされた時間にはCU料金は発生しません。

#### 例{#example}

クラスタ構成を次のようにします。

- **プラン:**専用（エンタープライズ）

- **クラウドプロバイダー&地域:**AWS us-east-1(Virginia)

- **CUのタイプ:**Performance-optimizedCU

- **CUのサイズ:**8 CU

- **レプリカ数:**2

- **クラスター** **ランタイム:**720時間(1か月)。

プラン、クラウドプロバイダーと地域、およびCUタイプ情報を使用すると、[価格ページ](https://zilliz.com/jp/pricing)でCUユニット価格が$0.248/時間であることがわかります。

![find-cu-unit-price](/img/ja-JP/find-cu-unit-price.png)

CU体格とレプリカ数の情報によると、CU数は8 CU x 2 Replica=16であることがわかります。

例の専用クラスターの合計CUコストは、$0.2 4 8 x 16 x 720=$2856.96です。

## サーバーレスクラスタ{#serverless-clusters}

サーバーレスクラスターは従量課金モデルで動作するため、高い弾力性を持つワークロードに最適です。

サーバーレスクラスターの総コストには、**書き込みコスト**、**読み取りコスト**、および**ストレージコスト**が含まれます。サーバーレスクラスターのストレージ単価は月額$0.300/GBで、ストレージコストは通常、合計の約2%を占めます。正確なコスト見積もりには、[計算機](https://zilliz.com/jp/pricing#calculator)を使用してください。

### 書き込みコスト{#write-cost}

書き込みコストは[、挿入、upsert、および削除操作](./insert-update-delete)によって消費されるコンピューティングリソースを測定します。

インポートおよび一括挿入操作には書き込みコストはかかりません。

#### コスト計算{#cost-calculation}

```bash
Write Cost = vCU Unit Price x Write vCU Usage 
```

- **vCUの単価:**100万vCUあたり4ドル

- **vCU使用状況:**書き込み操作に関与するデータ体格に基づいて計算されます

### 読み取りコスト{#read-cost}

読み取りコストは[、検索、ハイブリッド検索、およびクエリ操作](./search-query-get)で消費されるリソースを測定します。

#### コスト計算{#cost-calculation}

```bash
Read Cost = vCU Unit Price x Read vCU Usage 
```

- **vCUの単価:**100万vCUあたり4ドル

- **Read vCU U U sage:**スキャンされたデータと返されたデータの体格によって異なります。検索またはクエリのたびに、Zilliz Cloudはクラスタ内のコレクション全体をスキャンします。検索またはクエリ中に[パーティションキー](./use-partition-key)をフィルターとして使用する場合、Zilliz Cloudは指定されたパーティションキーに一致するコレクションの一部のみをスキャンするため、全体的な読み取りvCUの使用率が低下する可能性があります。

    <Admonition type="info" icon="📘" title="ノート">

    <ol>
    <li><p>vCUの使用量は、スキャンされたデータ体格に比例して増加しません。</p></li>
    <li><p>各読み取り操作には最低6つのvCUが必要です。</p></li>
    </ol>

    </Admonition>

## 付加価値サービス{#value-added-services}

Zilliz Cloudは、データバックアップやパイプラインなどの付加価値サービスも提供しています。

### バックアップコスト{#backup-cost}

バックアップとは、データが破損または失われた場合に復元に使用できるZilliz Cloudクラスターまたはコレクションのコピーです。

バックアップの作成には料金がかかりますが、バックアップファイルの復元には料金はかかりません。

#### コスト計算{#cost-calculation}

```plaintext
Backup Cost = Backup Unit Price x Backup File Size x Backup Rentention Period
```

- **バックアップ単価（月額）:**クラウドプロバイダーによって異なります。詳細については、[価格ページ](https://zilliz.com/jp/pricing)を参照してください。

- **バックアップファイルサイズ(GB):**バックアップファイルの合計サイズ。

- **バックアップ保持期間(月):**バックアップファイルが保持される期間を示します。

<Admonition type="info" icon="📘" title="ノート">

<p>バックアップは毎日課金されます。1日分の小なりがあるファイルは切り上げられ、1日分として課金されます。</p>

</Admonition>

#### 例{#example}

例えば、AWSがホストするクラスタを使用した場合、Zilliz Cloudに45日間（1.5ヶ月間）保存した20 GBのバックアップには、**$0.0 25 x 20 x 1.5=$0.75**がかかります。

### パイプラインのコスト{#pipelines-cost}

Zilliz Cloudに組み込まれているパイプラインを使用すると、1つのAPIサービスで埋め込み、ベクトルインジェスト、検索、および再ランキングを効率化できます。パイプラインのコストは、モデルの種類と実際の使用状況によって決定されます。詳細については、[パイプライン価格](https://zilliz.com/jp/pricing)を参照してください。
