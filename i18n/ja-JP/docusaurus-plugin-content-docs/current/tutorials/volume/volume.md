# ボリューム

ボリュームは、構造化データまたは非構造化データファイルのコレクションのいずれかを保持するオブジェクトストアです。これは、これらのデータ資産にアクセスし、保存し、管理し、整理するための統一された場所を提供します。この章では、ボリュームとは何か、その使用例、およびZilliz Cloudでボリュームを管理する方法について説明します。
