# Zilliz Cloudを無料で試す

Zilliz Cloudは、**無料クラスター**と**無料トライアル**の両方を提供し、強力なベクターデータベース機能の評価やテスト、Zilliz Cloud使用時のコストを算出するのに役立ちます。開始するには、Zilliz Cloudで[アカウントを登録](./register-with-zilliz-cloud)するだけです。支払い情報は必要ありません。

## 無料クラスター\{#free-cluster}

Zilliz Cloudは、基本的なベクターデータベース機能を無料で使用できる無料クラスターを提供します。無料クラスターでは、以下の機能を提供します：

- 5GBのストレージ（768次元ベクトル100万個分に十分）

- 月間250万vCU

- 最大5つのコレクション

より多くのリソースが必要な場合や、高度な機能へのアクセスを希望される場合は、当社のサーバーレスクラスターや専用クラスターの[無料トライアル](./free-trials#free-trial)をご利用ください。

## 無料トライアル\{#free-trial}

Zilliz Cloudは、サーバーレスクラスターと専用クラスターの無料トライアルを提供します。

### 無料トライアルの利用\{#use-free-trial}

Zilliz Cloudに仕事用メールアドレスでサインアップすると、組織には請求アカウントに追加される**$100**の無料クレジットが付与されます。これらのクレジットは**30日間**で期限が切れ、サーバーレスクラスターや専用クラスターの探索に使用できます。クレジットが使い切れたか期限切れになると、無料トライアルは終了します。

トライアル終了後、組織は凍結されます。この期間中、サーバーレスクラスターや専用クラスターは[ごみ箱](./use-recycle-bin)に移動され、これらのクラスターに特有の機能（バックアップと復元、アラートなど）にアクセスできなくなります。

組織の凍結を解除するには、[支払い方法を追加](./payment-billing)してください。これにより、ごみ箱から削除されたデータを復元できるようになります。凍結後30日以内に支払い方法を追加しない場合、サーバーレスクラスターや専用クラスターは完全に削除されますが、組織自体は維持されます。

### クレジットの獲得とクレジット期限の延長\{#earn-credits-and-extend-credit-expiration}

仕事用メールアドレスで登録すると、&#36;100の無料クレジットを受け取れます。[Zilliz Cloudで支払い方法を追加](./payment-billing)することで、さらに&#36;100を獲得できます。さらに、支払い方法を追加すると、クレジットの有効期限が**1年**に延長されます。

追加のクレジットが必要な場合や、トライアル期間を延長したい場合は、[営業担当までお問い合わせください](https://zilliz.com/contact-sales)。

### クレジット残高の確認\{#view-credit-balance}

左側のナビゲーションペインの下部で、残りのクレジットとその有効期限を素早く確認できます。

または、**請求概要**ページの**クレジット**セクションに移動してください。

### 無料トライアル通知\{#free-trial-notifications}

無料トライアル期間中、Zilliz Cloudからそのステータスに関する複数のメール通知が届きます。これらのメールは組織オーナー宛てに送信され、以下のイベントがきっかけで送信されます：

- クレジットが付与されてから最初の3日間にクレジットが使用されなかった場合。

- クレジットの60%が使用された場合。

- クレジットが3日以内に期限切れになる場合。

- 有効な支払い方法がないためにトライアル期限切れにより組織が凍結された場合。

- トライアル終了に伴い、サーバーレスクラスターや専用クラスターが削除されようとしている場合。

- トライアル終了後にサーバーレスクラスターや専用クラスターがごみ箱に移動された場合。

- すべてのクレジットが使い果たされた場合。
