# リリースノート (2023年1月10日)

Zilliz Cloudの重要なアップデートの一般提供を発表できることを嬉しく思います。このリリースでは、データプレビュ機能とコンソールでコレクションに簡単にインポートできるデモデータセットを追加しました。

- データプレビュー

    このリリースでは、Zilliz Cloudがデータプレビューキャパビリティを提供します。各コレクションページに**データプレビュー**機能タブを見つけることができます。プレビューテーブルを使用すると、コレクション内のサンプルエンティティのすべての値を表示できるため、データを簡単に検証できます。

    データプレビューキャパビリティに加えて、プレビューテーブルの任意のベクトルを選択してクエリベクトルとして使用できます。これにより、コレクション内の隣接データをすばやく検証できます。

- デモデータセット

    Zilliz Cloudは 이제、ベクトルデータベースの機能に慣れ親しむ初心者を支援するためのデモデータセットを提供します。デモデータセットはKaggleデータセットから派生しています。2020年1月から2020年8月にかけて著名な出版物で公開された5,000件以上の記事の詳細を含んでいます。

    これで、データ準備に余分な労力をかけずに、ベクトルデータベース機能のすべてを簡単に体験できます。[お楽しみください！](https://cloud.zilliz.com/signup)
