# リリースノート (2023年9月27日)

Zilliz Cloudのローンチを発表できることを嬉しく思います。このリリースでは、新しいCUサイズオプションによる柔軟性の向上、前払い支払いのサポート、およびより多くのAWSリージョンなど、さまざまな強化と機能を導入しています。

## Milvus互換性\{#milvus-compatibility}

このリリースは**Milvus 2.2.x**と互換性があります。

## クラスター強化 - 新CUサイズオプションによる柔軟性の向上\{#cluster-enhancements-increased-flexibility-with-a-new-cu-size-option}

クラスターマネジメントは、6 CUの新しいクラスターサイズオプションにより柔軟になりました。このオプションは、よりコスト効果が高く効率的なクラスターオプションを求める、较小なデータセットを持つユーザーに理想的です。特定のニーズに合わせて調整されたパフォーマンスの向上を体験してください。

## 前払い支払いのサポート\{#support-for-advance-pay}

前払い支払いのサポートを開始しました。この機能は、消費するサービスのために前払いを希望するオフライン契約ユーザー向けに特別に設計されています。このオプションを提供することで、顧客の貴重な内部財務および調達プロセスを合理化し、リスクを最小限に抑えることを目指しています。

ご自身または組織に特定の要件がある場合、または前払いオプションの詳細を知りたい場合は、[お問い合わせください](https://zilliz.com/contact-sales)をためらわずお知らせください。

## より多くのAWSリージョン\{#more-aws-regions}

aws-us-east-1（米国東部1）の新しいAWSリージョンをサポートすることを嬉しく思います。このリージョンの追加により、ユーザーはこのリージョンでのデータ処理能力を強化するために弊社のサービスを活用できます。
