# リリースノート（2023年9月27日）

Zilliz Cloudのローンチを発表できることを嬉しく思います。このリリースでは、新しいCUサイズオプションによる柔軟性の向上、前払いのサポート、より多くのAWSリージョンなど、さまざまな強化と機能を導入しています。

## Milvusの互換性{#milvus-compatibility}

このリリースは**Milvus 2.2. x**と互換性があります。

## クラスタの強化–新しいCUサイズオプションによる柔軟性の向上{#cluster-enhancements-increased-flexibility-with-a-new-cu-size-option}

クラスター管理は、新しいクラスター体格オプションである6 CUにより、より柔軟になりました。このオプションは、より費用対効果が高く効率的なクラスターオプションを探している小規模なデータセットを持つユーザーに最適です。特定のニーズに合わせて改善されたパフォーマンスを体験してください。

## 事前支払いのサポート{#support-for-advance-pay}

私たちは現在、前払いのサポートを提供しています。この機能は、消費するサービスの前払いを好むオフライン契約ユーザー向けに特別に設計されています。このオプションを提供することで、内部の財務および調達プロセスを効率化し、貴重なお客様のリスクを最小限に抑えることを目指しています。

特定の要件がある場合、または前払いオプションについて詳しく知りたい場合は、お気軽に[お問い合わせ](https://zilliz.com/contact-sales)ください。

## その他のAWSリージョン{#more-aws-regions}

私たちは、新しいAWSリージョンであるUS East 1(aws-us-east-1)をサポートすることを発表できることを嬉しく思います。このリージョンの追加により、ユーザーはこのリージョンでのデータ処理能力を向上させるために私たちのサービスを利用することができます。

## その他のハイライト{#other-highlights}

- クレジット有効期限の監視機能のUIを強化しました

- プレイグラウンド機能でのユーザーエクスペリエンスの向上
