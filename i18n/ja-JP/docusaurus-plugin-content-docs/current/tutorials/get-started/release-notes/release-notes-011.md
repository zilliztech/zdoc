# リリースノート (2023年2月13日)

Zilliz Cloudの重要なアップデートの一般提供を発表できることを嬉しく思います。このリリースでは、メール通知機能と初心者のためのインラインガイダンスを追加しました。

- メール通知

    このリリースでは、メール通知機能が強化されています。以下のイベントでメール通知を受信します：

    - アカウントの確認、停止、再開など

    - クレジット残高および有効期限

    - 請求および支払い

    - データベースの非アクティブ状態

    - 近づくシステムメンテナンス

    - 登録されたリソースアラート

    - ユーザーフィードバック受領の確認

- 初心者のためのインラインガイダンス

    このリリースでは、初心者のためのインラインガイダンスが大幅に改善され、迅速なオンボーディングが可能になりました。今や、背景知識がなくても5分以内にオンラインチュートリアルを完了できます。[お楽しみください！](https://cloud.zilliz.com/signup)
