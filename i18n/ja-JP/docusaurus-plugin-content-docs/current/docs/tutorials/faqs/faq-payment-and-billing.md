## FAQs




### クレジットカードを追加した後、アカウントに残っている期限切れのクレジットを引き続き使用できますか?{#after-adding-a-credit-card-can-i-continue-to-use-the-expired-credits-left-in-my-account}

はい。アカウントにまだ残っているクレジットがあり、支払い方法を追加した場合、残りのクレジットの有効期限は1年間延長されます。

### $100のクレジットは期限切れになりますか?無料のクレジットの期限を延長するにはどうすればよいですか?{#will-my-dollar100-credits-expire-how-can-i-extend-the-expiration-of-the-free-credits}

最初の登録から3 0日以内に使える100ドルのクレジットを受け取ります。クレジットの有効期限が切れる場合は、Zilliz Cloud UIで支払い方法を追加することで有効期限を延長できます。クレジットカードを追加する場合、クレジットの有効期限は1年間に延長されます。支払い方法としてMarketplaceサブスクリプションを選択した場合、クレジットの有効期限も1年間に延長され、有効期限が1年の100ドルの追加クレジットを獲得できます。

### ごみ箱のデータは課金されますか?{#is-data-in-the-recycle-bin-charged}

いいえ。ごみ箱から削除されたデータについては料金はかかりませ ん。
