# データインポート

Zilliz Cloudは、AWS S 3、GCP Storage、Azure Blob Storageなど、さまざまなストレージオプションからのデータインポートをサポートしています。また、Parquet(推奨)、JSON、NumPyファイルなどの形式もサポートしています。この章では、これらのストレージオプションからサポートされている形式のデータをインポートする手順について説明します。
