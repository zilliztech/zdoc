# マイグレーション

Zilliz Cloudは、Milvus、Elasticsearch、Postgre SQLなどの外部ソースからの移行オプションを提供し、同じ組織内または異なる組織内でのクラスタ間の移行を可能にし、効率的なデータ統合と管理を可能にします。
