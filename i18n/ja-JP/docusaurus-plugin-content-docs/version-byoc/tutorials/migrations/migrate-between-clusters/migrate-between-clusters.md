# ZillizからZillizへの移行

ZillizからZillizへの移行により、同じ組織内または異なる組織に存在するZilliz Cloudクラスター間でデータを転送できます。主なユースケースは以下のとおりです：

- 組織間で安全にデータを共有。

- パフォーマンス最適化のためのインフラの再編成。
